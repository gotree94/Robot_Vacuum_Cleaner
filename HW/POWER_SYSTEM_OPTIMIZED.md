## 설계 요구사항

### 목표 사양

| 항목 | 요구사항 | 우선순위 |
|-----|---------|:-------:|
| 사용시간 | 180분 (에코모드) / 90분 (일반) | 필수 |
| 충전시간 | 3시간 이내 | 필수 |
| 어댑터 | DC 24V 입력 | 필수 |
| 본체 높이 | 98mm 이하 유지 | 필수 |

### 동작 모드별 소비 전력

| 모드 | 소비 전력 | 목표 시간 | 필요 에너지 |
|-----|--------:|:-------:|----------:|
| 에코 (Eco) | 29W | 180분 | 87Wh |
| 일반 (Normal) | 55W | 90분 | 82.5Wh |
| 강력 (Turbo) | 95W | 45분 | 71.25Wh |

→ **최소 배터리 용량: 87Wh × 1.1(마진) = 약 96Wh**

## 배터리 사양 결정

### 18650 셀 vs 21700 셀 비교

| 항목 | 18650 | 21700 |
|-----|:-----:|:-----:|
| 크기 | Ø18 × 65mm | Ø21 × 70mm |
| 용량 | 2,600~3,500mAh | 4,000~5,000mAh |
| 무게 | 45~48g | 65~70g |
| 단가 | 2,800원 | 4,500원 |
| 에너지 밀도 | 250Wh/kg | 270Wh/kg |

| 사용시간 | 에코 199분 / 일반 105분 |

## 어댑터 사양 결정

### 충전 요구사항

```
배터리 용량: 129.5Wh
목표 충전시간: 3시간
필요 충전전력: 129.5Wh ÷ 3h ÷ 0.85(효율) = 50.8W

24V 어댑터 필요 전류: 50.8W ÷ 24V = 2.12A
```

### 어댑터 옵션

| 사양 | 전력 | 충전시간 | 원가 | 판정 |
|-----|:----:|:------:|-----:|:---:|
| 24V / 1.25A | 30W | 5.1시간 | 3,500원 | ❌ 너무 느림 |
| 24V / 1.5A | 36W | 4.3시간 | 4,000원 | △ |
| **24V / 2.0A** | **48W** | **3.2시간** | **4,500원** | **✅ 권장** |
| 24V / 2.5A | 60W | 2.6시간 | 5,500원 | ○ 고속충전 |

### ✅ 최종 어댑터 사양 결정

| 항목 | 사양 |
|-----|------|
| **입력** | AC 100-240V, 50/60Hz |
| **출력** | DC 24V / 2.0A |
| **출력 전력** | 48W |
| **효율** | 87% 이상 (Level VI) |
| **커넥터** | DC 잭 Ø5.5 × 2.1mm |
| **안전 인증** | KC, UL, CE |
| **케이블 길이** | 1.5m |
