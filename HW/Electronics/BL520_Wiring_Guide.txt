# BL520 Charger Board - KiCad 배선 가이드
# 이 파일은 KiCad에서 배선할 때 참고용입니다.

===============================================================================
                         NET LABEL 배선 가이드
===============================================================================

KiCad에서 부품을 배치한 후, 아래 Net Label을 사용하여 연결하세요.
Net Label은 같은 이름끼리 자동으로 연결됩니다.

-------------------------------------------------------------------------------
NET: VIN (DC 입력 전원 18-24V)
-------------------------------------------------------------------------------
연결할 핀:
  - DC1.Pin1 (DC Jack VIN+)
  - D1.Anode
  - D2.Anode

-------------------------------------------------------------------------------
NET: VIN_FILT (필터링된 입력 전원)
-------------------------------------------------------------------------------
연결할 핀:
  - D1.Cathode
  - D2.Cathode  
  - L1.Pin1
  - L2.Pin1

-------------------------------------------------------------------------------
NET: VIN_REG (레귤레이터 입력)
-------------------------------------------------------------------------------
연결할 핀:
  - L1.Pin2
  - L2.Pin2
  - C2.+ (100uF)
  - C3.+ (10uF)
  - C5.+ (100uF)
  - C4.+ (10uF)
  - U1.VI (Pin 3)

-------------------------------------------------------------------------------
NET: +5V (5V 전원 레일)
-------------------------------------------------------------------------------
연결할 핀:
  - U1.VO (Pin 2)
  - C6.+ (10uF)
  - C7.Pin1 (100nF)
  - LED1.Anode (IR RX)
  - LED2.Anode (IR TX)
  - LED3.Anode (IR RX)
  - LED4.Anode (IR TX)
  - LED5.Common (Pin 1)
  - J1.Pin1 (FPC VCC)

-------------------------------------------------------------------------------
NET: GND (접지)
-------------------------------------------------------------------------------
연결할 핀:
  - DC1.Pin2 (DC Jack GND)
  - C2.- 
  - C3.-
  - C5.-
  - C4.-
  - C6.-
  - C7.Pin2
  - U1.GND (Pin 1)
  - Q1.E, Q3.E, Q5.E, Q7.E, Q9.E (SOT-89 Emitter)
  - Q2.E, Q4.E, Q6.E, Q8.E, Q10.E, Q11.E (SOT-23 Emitter)
  - LED1.Cathode, LED3.Cathode (IR RX)
  - J1.Pin2 (FPC GND)
  - CHARGE.Pin2

-------------------------------------------------------------------------------
NET: IR_TX1 (LED2 → Q1)
-------------------------------------------------------------------------------
연결할 핀:
  - LED2.Cathode
  - Q1.Collector

-------------------------------------------------------------------------------
NET: IR_TX2 (LED4 → Q3)
-------------------------------------------------------------------------------
연결할 핀:
  - LED4.Cathode
  - Q3.Collector

-------------------------------------------------------------------------------
NET: Q1_BASE
-------------------------------------------------------------------------------
연결할 핀:
  - R11.Pin2
  - Q1.Base

-------------------------------------------------------------------------------
NET: Q3_BASE
-------------------------------------------------------------------------------
연결할 핀:
  - R12.Pin2
  - Q3.Base

-------------------------------------------------------------------------------
NET: Q5_BASE
-------------------------------------------------------------------------------
연결할 핀:
  - R13.Pin2
  - Q5.Base

-------------------------------------------------------------------------------
NET: Q7_BASE
-------------------------------------------------------------------------------
연결할 핀:
  - R14.Pin2
  - Q7.Base

-------------------------------------------------------------------------------
NET: Q9_BASE
-------------------------------------------------------------------------------
연결할 핀:
  - R15.Pin2
  - Q9.Base

-------------------------------------------------------------------------------
NET: CHARGE_OUT (충전 출력)
-------------------------------------------------------------------------------
연결할 핀:
  - CHARGE.Pin1
  - (DC 입력에서 분기)

-------------------------------------------------------------------------------
NET: FPC_SIG (플렉서블 PCB 신호)
-------------------------------------------------------------------------------
연결할 핀:
  - J1.Pin3

===============================================================================
                         KiCad 배선 방법
===============================================================================

방법 1: Net Label 사용 (권장)
------------------------------
1. KiCad Eeschema에서 회로도 열기
2. 우측 도구바에서 "Place net label" (L 키) 선택
3. 각 핀 근처에 위의 Net 이름으로 라벨 배치
4. 같은 이름의 라벨은 자동으로 연결됨

방법 2: Wire로 직접 연결
------------------------------
1. 우측 도구바에서 "Place wire" (W 키) 선택
2. 핀에서 핀으로 클릭하여 배선
3. 교차점에서 Junction 추가 (J 키)

방법 3: Global Label 사용
------------------------------
전원(+5V, GND, VIN)은 Global Label 사용 권장
1. "Place global label" (Ctrl+L) 선택
2. +5V, GND 등 전원 심볼 배치

===============================================================================
                         트랜지스터 연결 상세
===============================================================================

SOT-89 (Q1, Q3, Q5, Q7, Q9) - 고전류 IR LED 드라이버
-----------------------------------------------------
Pin 1: Base    ← R11~R15 (1K)
Pin 2: Collector → IR LED Cathode
Pin 3: Emitter  → GND

SOT-23-3 (Q2, Q4, Q6, Q8, Q10, Q11) - 신호 제어
-----------------------------------------------------
Pin 1: Emitter  → GND
Pin 2: Base     ← 제어 신호
Pin 3: Collector → 출력

LED5 (Green 2-Color Milky) - 3핀
-----------------------------------------------------
Pin 1: Common Anode → +5V
Pin 2: Green1 Cathode → Q_control1
Pin 3: Green2 Cathode → Q_control2
