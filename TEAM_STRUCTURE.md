## 분야별 상세 인력 구성

### 1. 경영/관리

#### CEO (Chief Executive Officer) - 1명

| 항목 | 내용 |
|-----|------|
| **역할** | 전사 경영 총괄, 투자 유치, 전략적 의사결정 |
| **주요 업무** | 비전/전략 수립, 이사회 운영, 외부 파트너십, 주요 의사결정 |
| **필요 경력** | 10년+ (제조업/스타트업 경영 경험) |
| **필요 역량** | 리더십, 전략적 사고, 네트워킹, 투자 유치 경험 |
| **학력** | 학사 이상 (MBA 우대) |

#### CTO (Chief Technology Officer) - 1명

| 항목 | 내용 |
|-----|------|
| **역할** | 기술 총괄, R&D 전략, 기술 로드맵 수립 |
| **주요 업무** | 기술 방향성 결정, 아키텍처 리뷰, 핵심 기술 의사결정, 기술 조직 관리 |
| **필요 경력** | 15년+ (HW/SW 통합 제품 개발 경험) |
| **필요 역량** | 시스템 아키텍처, 임베디드 시스템, IoT, 기술 리더십 |
| **학력** | 석사 이상 (전자/컴퓨터공학) |
| **우대사항** | 로봇/가전 제품 개발 경험, 스타트업 CTO 경험 |

#### Project Manager (PM) - 1명

| 항목 | 내용 |
|-----|------|
| **역할** | 제품 개발 프로젝트 총괄 관리 |
| **주요 업무** | 일정/리소스 관리, 이슈 트래킹, 팀간 조율, Gate Review 주관 |
| **필요 경력** | 7년+ (HW 제품 개발 PM 경험) |
| **필요 역량** | PMP/Agile, 리스크 관리, 커뮤니케이션, 일정 관리 도구 |
| **학력** | 학사 이상 (이공계) |
| **우대사항** | 가전/로봇 제품 PM 경험, PMP 자격증 |
